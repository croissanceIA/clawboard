# Tailwind Color Configuration

## Color Choices

- **Primary:** `blue` — Used for buttons, links, active states, key accents
- **Secondary:** `amber` — Used for cost badges, warning states, highlights
- **Neutral:** `zinc` — Used for backgrounds, text, borders, cards

## Usage Examples

Primary button: `bg-blue-600 hover:bg-blue-700 text-white`
Primary ghost: `text-blue-600 hover:bg-blue-50 dark:text-blue-400`
Secondary badge: `bg-amber-100 text-amber-700 dark:bg-amber-900/40 dark:text-amber-400`
Neutral text: `text-zinc-600 dark:text-zinc-400`
Neutral border: `border-zinc-200 dark:border-zinc-800`
Card background: `bg-white dark:bg-zinc-900`
Page background: `bg-white dark:bg-zinc-950`

## Status Colors

- Success: `emerald` — `bg-emerald-100 text-emerald-700`
- Error: `red` — `bg-red-100 text-red-700`
- Warning: `amber` — `bg-amber-100 text-amber-700`
- Info: `blue` — `bg-blue-100 text-blue-700`
